## Development

### Environment
- Node.js >= 18 (required)
- Visual Studio Code または他のエディタ

### CLI Commands

```bash
# 依存関係のインストール
> npm install

# コードの記法チェック (use ESLint & Prettier)
> npm run lint
# コードの記法チェック & 自動修正
> npm run fix

# 開発サーバーの起動 (see https://nuxt.com/docs/api/commands/dev)
> npm run dev

# 本番ビルド
> npm run build
# 本番ビルドした内容のプレビュー
> npm run preview
```
